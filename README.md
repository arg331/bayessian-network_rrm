### The Bayessian Network  
Un servidor web recibe cada hora una media de 1000 peticiones HTTP, de las cuales se estima
que 200 son **ataques**. Se estima que el 50 % de los ataques pueden suponer **un fallo grave** para
la integridad del sistema. Se instala un **cortafuegos** que, según especificaciones del fabricante, no
conseguirá bloquear 1 de cada 4 ataques. Además, dicho cortafuegos se **actualizará** una vez cada
hora, lo cual tardará 6 minutos y durante este tiempo no estará activo. Por otro lado, se instalará
un **sistema de recuperación** *que estará en funcionamiento el 85 % del tiempo* y que evitará la mitad
de los fallos graves.
